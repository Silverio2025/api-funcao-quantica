# Quantum Translator

SaaS de tradução automática com inteligência artificial e computação quântica (PennyLane).  
Traduz textos para múltiplos idiomas e aplica otimizações quânticas personalizadas.

## Tecnologias

- Python + Flask
- PennyLane
- OpenAI API
- Render.com (deploy)
- Emergent.sh (interface no-code)
